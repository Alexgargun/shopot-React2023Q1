## Installation

```bash
$ git clone https://github.com/shopot/shopot-React2023Q1.git
$ cd shopot-React2023Q1
$ git checkout react-forms
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build production
$ npm run build

# preview mode
$ npm run preview

# start linting
$ npm run lint
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run coverage
```
